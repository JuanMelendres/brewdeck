# Accessibility Review Checklist

## Structure

- [ ] Unique page title
- [ ] Logical heading hierarchy
- [ ] Landmarks
- [ ] Lists use list semantics
- [ ] Tables use captions and headers
- [ ] Forms use labels
- [ ] Related fields use fieldset and legend

## Keyboard

- [ ] All actions keyboard accessible
- [ ] Logical tab order
- [ ] Visible focus
- [ ] No keyboard trap
- [ ] Dialog Escape behavior
- [ ] Focus returns after close
- [ ] Skip link when needed

## Forms and errors

- [ ] Required fields communicated
- [ ] Errors associated with fields
- [ ] Error summary when appropriate
- [ ] Focus moves to error summary or first error
- [ ] Values preserved after failure
- [ ] Submission status announced

## Dynamic content

- [ ] Loading announced when useful
- [ ] Success announced
- [ ] Errors announced
- [ ] Pagination or filter updates announced
- [ ] No excessive live-region noise

## Visual

- [ ] Text contrast
- [ ] Non-text contrast
- [ ] No color-only meaning
- [ ] Zoom and text resizing
- [ ] Focus indicator
- [ ] Touch target size
- [ ] Reduced motion
- [ ] Long text and localization

## Media and icons

- [ ] Informative images have alternatives
- [ ] Decorative images are ignored
- [ ] Icon-only controls have accessible names
- [ ] Remote-image failure state

## Manual review

- [ ] Keyboard-only
- [ ] Screen reader
- [ ] 200% zoom
- [ ] Reflow
- [ ] Reduced motion
- [ ] High contrast or forced colors where supported
